===============
Getting started
===============

``saspt`` uses Python 3 and you'll need to know a little Python.

Install
=======

``saspt`` can be installed with ``pip``:

.. code-block:: bash

    pip install saspt

``saspt`` requires the following Python modules:
    * ``numpy``
    * ``dask``
    * ``pandas``
    * ``matplotlib``
    * `tqdm <https://tqdm.github.io/>`_
    * ``seaborn``
